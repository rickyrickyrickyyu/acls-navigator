# Contributing

感謝您對 ACLS Navigator 的關注！

## 如何貢獻

1. Fork 本專案
2. 建立 feature branch (`git checkout -b feature/your-feature`)
3. 提交變更 (`git commit -m 'feat: add your feature'`)
4. Push 到 branch (`git push origin feature/your-feature`)
5. 開啟 Pull Request

## 開發注意事項

- 本專案為 **單一 HTML 檔案**（`index.html`），所有 CSS 和 JS 都內嵌
- 修改後請在以下裝置/尺寸測試：
  - 手機（375px 寬）
  - 平板（768px 寬）
  - 桌面（1200px 寬）
- 請確保深色模式和淺色模式都正常顯示

## 醫療內容修改

涉及藥物劑量、流程步驟等醫療內容的修改，請附上參考文獻來源。

## 回報問題

請使用 [GitHub Issues](https://github.com/rickyrickyrickyyu/acls-navigator/issues) 回報問題或建議功能。
